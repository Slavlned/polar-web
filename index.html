Why Polar?

Polar is designed to simplify programming experience and provide powerfull tools. Polar is dynamically typed, what makes writing code easier. Easy syntax to learn and use. Here is example of multithreading.


use 'lib.tasks'

func powerfull_thread1() = {
    put('Some thread!')
}

func powerfull_thread2() = {
    put('Some thread 2!')
}

Tasks.exec(powerfull_thread1, [])
Tasks.exec(powerfull_thread2, [])
        

put('Hello from Polar!')
